## 🛠️ 部署

#### 1. 环境（参考开发环境）

Nginx/Apache  
PHP8+  
Composer  
mysql

#### 2. TP6 安装依赖（发行版跳过）

`composer install`

#### 3. 配置

1. 设置运行目录为“Public”
2. 设置伪静态，伪静态规则参考“Public”目录下的“nginx.htaccess”与“nginx.htaccess”
3. 删除“lock.txt”安装记录文件
4. 赋予程序根目录操作权限为“777（出现异常时可选）
5. 关闭防跨站(宝塔可选)

#### 4. 进入网站自动跳转至安装引导
